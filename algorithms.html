Algorithms

Jump Search

A searching algorithm for sorted arrays that checks fewer elements than Linear Search by jumping ahead in fixed steps.

Interpolation Search

An improvement over Binary Search for uniformly distributed sorted arrays, where the probe position is calculated based on the target value.

Shell Sort

An efficient, in-place comparison sort that is a generalization of insertion sort, allowing the exchange of items far apart.

Heapsort

A comparison-based sorting algorithm that uses a binary heap data structure to build a max-heap or min-heap.

Radix Sort

A non-comparison integer sorting algorithm that sorts data by processing individual digits from least significant to most significant.

Kruskal's Algorithm

An algorithm to find a Minimum Spanning Tree (MST) for a connected, weighted, undirected graph, by repeatedly adding the next-cheapest edge that doesn't form a cycle.

Prim's Algorithm

Another algorithm to find a Minimum Spanning Tree (MST) for a connected, weighted, undirected graph, by growing a tree from an arbitrary starting vertex.

A* Search Algorithm

An informed search algorithm used for pathfinding that seeks to find the shortest path from a start node to a goal node, using a heuristic function.

Floyd-Warshall Algorithm

An algorithm for finding the shortest paths between all pairs of vertices in a weighted graph.

Knapsack Problem (0/1 Dynamic Programming)

A dynamic programming approach to solve the problem of selecting items with given weights and values to maximize the total value, within a given capacity.
